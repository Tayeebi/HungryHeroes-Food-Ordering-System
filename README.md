## ğŸš€ Getting Started

Follow these simple steps to get a local copy up and running.

### Prerequisites

* Java Development Kit (JDK) 17 or higher
* Git

### Installation

1.  **Clone the repo**
    ```bash
    git clone [https://github.com/YOUR-USERNAME/hungry-heroes.git](https://github.com/YOUR-USERNAME/hungry-heroes.git)
    ```
2.  **Navigate to the project directory**
    ```bash
    cd hungry-heroes
    ```
3.  **Compile the source code**
    ```bash
    # Create bin directory
    mkdir bin
    
    # Compile
    javac -d bin src/classes/Start.java src/classes/*.java src/interfaces/*.java
    ```
4.  **Run the application**
    ```bash
    java -cp bin classes.Start
    ```

> **Note:** The application uses relative paths for file storage. Ensure you run the command from the root project directory.
